RSA_sensor

Sistema de registro acelerométrico y transmisión MQTT

Repositorio

Resumen

RSA_sensor es un sistema embebido desarrollado sobre ESP32 para registrar aceleraciones en tiempo real mediante un acelerómetro ADXL355. Incorpora sincronización temporal por GPS, almacenamiento en tarjeta SD y transmisión eficiente de eventos por MQTT, con formatos JSON o binario. Gestiona tareas concurrentes mediante FreeRTOS y ofrece soporte para eventos bajo demanda con confirmaciones en tiempo real.
